# HR-Analytics-Dashboard
Interactive HR Analytics Dashboard built using Power BI to analyze employee attrition, salary trends, and workforce insights with dynamic visualizations and KPI tracking.

## Project Overview
This project analyzes employee data to identify attrition trends, salary distribution, and workforce insights using Power BI.

## Key Features
- KPI metrics: Total Employees, Attrition Rate, Average Salary
- Attrition analysis by department and experience
- Gender diversity insights
- Salary comparison across departments
- Interactive filters (slicers)

## Tools Used
- Power BI
- Microsoft Excel

## Key Insights
- High attrition among employees with 1–3 years of experience
- Sales department shows higher employee turnover
- IT and Finance departments have higher salary distribution
